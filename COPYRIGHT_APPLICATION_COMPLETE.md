## SECTION 8: IMPORTANT NOTES AND GUIDELINES

### 1. Copyright Protection Scope

**What Copyright Protects:**
- ‚úÖ The actual source code (literal expression)
- ‚úÖ The structure, sequence, and organization of the code
- ‚úÖ The user interface design and layout
- ‚úÖ The database schema and structure
- ‚úÖ The documentation and comments
- ‚úÖ The algorithms as expressed in code
- ‚úÖ The API endpoint structure
- ‚úÖ The configuration files

**What Copyright Does NOT Protect:**
- ‚ùå The underlying idea or concept
- ‚ùå The method or system (use patent for this)
- ‚ùå The functionality or features
- ‚ùå The programming language itself
- ‚ùå The algorithms in abstract form
- ‚ùå The user interface functionality (only visual expression)

### 2. Copyright Duration

**For Computer Software:**
- Duration: Lifetime of author + 60 years
- For joint authorship: Lifetime of last surviving author + 60 years
- Starts from: Year of creation (2024)
- Expires: Approximately 2084-2144 (depending on authors' lifetimes)

### 3. Copyright vs Patent

**File BOTH for Maximum Protection:**

| Aspect | Copyright | Patent |
|--------|-----------|--------|
| **Protects** | Source code | Method/System |
| **Cost** | ‚Çπ500 | ‚Çπ9,700 |
| **Duration** | Life + 60 years | 20 years |
| **Registration** | Optional (automatic) | Mandatory |
| **Prevents** | Code copying | Method copying |
| **Filing Time** | Anytime | Before public disclosure |

**Recommendation:** File copyright for code + patent for invention

### 4. Copyright Infringement

**Your Rights as Copyright Owner:**
- Sue for damages
- Seek injunction (stop infringement)
- Claim profits made by infringer
- Criminal prosecution (imprisonment + fine)

**What Constitutes Infringement:**
- Copying source code
- Creating derivative works without permission
- Distributing copies without authorization
- Reverse engineering and replicating
- Using substantial portions of code
- Creating "inspired by" versions with similar structure

### 5. Fair Use Exceptions

**Limited use allowed without permission:**
- Educational purposes (teaching, research)

**Note:** Fair use is very limited for software
