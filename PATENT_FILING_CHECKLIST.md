## üåç INTERNATIONAL PROTECTION

### PCT (Patent Cooperation Treaty):

**Advantages:**
- Single application covers 150+ countries
- 30-month window to enter national phase
- Centralized search and examination
- Cost-effective for multiple countries

**Process:**
1. File India patent (done)
2. File PCT within 12 months
3. Receive International Search Report
4. Receive International Preliminary Examination Report
5. Enter national phase in selected countries (within 30 months)

**Recommended Countries:**
- USA (largest market)
- China (manufacturing hub)
- Europe (EU patent covers 38 countries)
- UAE (Middle East hub)
- Singapore (ASEAN hub)

**Cost:** $10,000-50,000 total (depending on countries)
